🚀 An AI-powered Resume Builder that helps users create professional, ATS-friendly resumes in minutes. Users can enter their details, receive AI-generated suggestions, choose templates, and download resumes as PDF files.
📌 Features
🔐 User Authentication (Login & Signup)
📝 Create and Edit Resume
🤖 AI-Powered Content Suggestions
🎨 Multiple Resume Templates
📄 Export Resume as PDF
💾 Cloud Storage using Firebase
📱 Responsive Design
🌙 Dark Mode Support (Optional)
